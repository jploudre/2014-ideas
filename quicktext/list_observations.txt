This quicktext is a template for listing the last 3 results for a patient. (Such as, the last 3 A1cs.)

o observations array
t text to return
r observation row

{fn LObs(){local i, j, o=getfield(List_Obs("HDL","signed","delimited","value"),"|",""), t="HDL Trend"+HRET, r
j = Size(o)+1
for i=1, i < j, i=i+1
do
r=getfield(o[i],"^","")
t=t+r[1]+" "+ObsUnit("HDL")+"\t"+r[2]+HRET
endfor
return t
}}
{LObs()}
