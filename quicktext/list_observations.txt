This quicktext is a template for listing the last 3 results for a patient. (Such as, the last 3 A1cs.)

o observations array
t text to return
r observation row

{LObs("HDL","HDL is 'Healthy' Cholesterol. Target: more than 40")}
{fn LObs(){local i, j, o=getfield(List_Obs(getarg(1),"signed","delimited","value"),"|",""), t=getarg(2)+HRET, r
j = Size(o)+1
if j=1, return ""
for i=1, i < j, i=i+1
do
r=getfield(o[i],"^","")
t=t+r[1]+" "+ObsUnit(getarg(1))+"\t"+r[2]+HRET
endfor
return t
}}
